 #AI-Powered Resume Screening and Ranking System

## Overview
The **AI-Powered Resume Screening and Ranking System** is designed to automate the process of evaluating resumes using machine learning and natural language processing (NLP). This system helps HR teams and recruiters by analyzing, screening, and ranking candidates based on job requirements, skills, and experience, reducing manual effort and improving hiring efficiency.

## Features
- **AI-driven Resume Parsing**: Extracts key information such as skills, experience, education, and certifications.
- **Smart Candidate Ranking**: Assigns scores based on job descriptions and candidate qualifications.
- **Keyword Matching & NLP Processing**: Ensures relevant resumes are prioritized using semantic analysis.
- **Automated Pre-screening**: Filters out resumes that donâ€™t meet minimum job requirements.
- **Customizable Criteria**: Allows recruiters to define job-specific ranking metrics.
- **Dashboard & Visualization**: Provides insights on candidate suitability and comparison.

## Technologies Used
- **Programming Language**: Python
- **Machine Learning**: Scikit-learn
- **Data Processing**: Pandas


## Usage
- Upload resumes in PDF/DOCX format.
- Input job descriptions.
- View ranked candidates with scores and insights.
- Adjust ranking criteria if needed.
